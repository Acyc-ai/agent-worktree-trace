# Contributing to Agent Worktree Trace

Thank you for your interest in contributing! This project welcomes contributions from People and Agents (supervised) alike.

## License Agreement

By submitting a pull request, you agree that your contribution will be licensed under the [MIT License](LICENSE) that covers this project.

## How to Contribute

### Reporting Issues

- Check existing issues before creating a new one
- Include VS Code version, OS, and steps to reproduce
- For bugs, include relevant error messages or screenshots

### Pull Requests

1. Fork the repository
2. Create a branch (`git checkout -b [feat/fix]/my-feature`)
3. Make your changes
4. Run tests (`npm test`)
5. Commit with a clear message
6. Push and open a PR

### Code Style

- Follow existing code patterns (or make them better)
- Add tests for new functionality (I, nor Claude, want to do them for you)
- Keep changes atomic and per-commit (or branch at the very least)

## AI Agent Contributions

Contributions made with the assistance of AI agents are welcome. When submitting AI-assisted work:

- Review generated code before submitting
- Ensure tests pass (and make sense)
- You remain responsible for the contribution's quality and correctness

## Development Setup

```bash
# Install dependencies
npm install

# Build
npm run compile

# Run tests
npm test

# Package extension
npm run package
```
